# photogal

A photo gallery web application.

## Running Locally

Run `make init`, which creates a virtual environment in a new `venv` directory. Then use the `photogal.sh` script
to run the app. 

````
$ make init
$ ./photogal.sh
````

## Building a Distribution

````
$ make envclean init dist
````
